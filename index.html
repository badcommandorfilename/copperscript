Copperscript

Copperscript is a Script# import library for the CopperLicht WebGL library. Alpha version - only a subset of the CopperLicht API is available.

Download this project as a .zip file Download this project as a tar.gz file

copperscript (alpha)

Please note: Copperscript is still a work in progress. Only a small section of the CopperLicht API has been imported.

Introduction

Copperscript is a wrapper for the CopperLicht WebGL Library so that it can be used with Script Sharp.
